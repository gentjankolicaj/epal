## Technology Stack

1. Java 11
2. Spring boot (DATA JPA , SECURITY , CLOUD)

4. Slf4j & Lombok & Mapstruct
5. Apache (Commons,HttpCore, HttpClient)
6. MySQL , REDIS , MONGODB
7. Maven
8. Docker
